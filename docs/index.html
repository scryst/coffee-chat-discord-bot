Getting Started

  1. Add the bot to your server using the "Add to Discord" button above
  2. Use the /coffee command to open the main menu
  3. Select "Request Coffee Chat" to create your first request
  4. Fill out the topic and description for what you'd like to discuss
  5. Wait for someone to accept your request, or browse existing requests to accept one
  6. When connected, chat through the bot's DMs
  7. End the chat when finished using the "End Chat" button

Commands

The bot uses a single slash command with a menu-based UI:

  • /coffee - Opens the main menu with buttons for all functionality

For Server Owners

Coffee Chat Bot requires the following permissions:

  • Read Messages/View Channels
  • Send Messages
  • Embed Links
  • Attach Files
  • Use Slash Commands
  • Read Message History

The bot also requires the "Message Content" and "Server Members" intents to function properly.
